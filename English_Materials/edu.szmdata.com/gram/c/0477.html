3.67 ergative verbs which need adjuncts

There are a small number of ergative verbs which usually have an adverb or other adjunct when they are used intransitively. This is because you choose this structure when you want to emphasize how something behaves when affected in some way, and so the person who does the action is not important.

  • I like the new Range Rover. It handles beautifully.
  • Wool washes well if you treat it carefully.

Here is a list of ergative verbs which usually have an adjunct when they are used intransitively:

  • clean, handle, mark, polish, sell, stain, wash
